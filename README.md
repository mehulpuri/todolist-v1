# todolist-v1

todolist app made using EJS and express

## Deployed App

Click the link to visit the [App](https://memu-blogwebsite.herokuapp.com/)

## Instructions
To build the app
```bash
$ npm i
```
and then use

```bash
node app.js 
```
OR
```bash
nodemon app.js 
```
to run the app
